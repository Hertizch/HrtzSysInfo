# HrtzSysInfo

Displays the following information, in a customizable overlay, on your desktop:
- CPU usage
- RAM usage
- Harddrive usage (multiple)
- CPU temperature
- GPU tempearture (multiple)
- Internal IP address
- External IP address
- Network usage (up/down)
- Clock, date and week
